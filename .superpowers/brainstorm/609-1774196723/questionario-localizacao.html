Onde o questionário vai viver no site?

Escolha onde o cliente vai encontrar o formulário

A

Dentro da página de Contato

O formulário substitui ou complementa os links atuais (WhatsApp, e-mail, Instagram) na /contato

B

Página separada: /questionario

O formulário ganha sua própria rota. A página de Contato direciona o cliente para ela via botão ou link

C

Modal / overlay

Um botão em qualquer página abre o questionário como uma sobreposição, sem sair da navegação atual
